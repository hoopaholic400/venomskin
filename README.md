# Venom — venomskin.com (concept draft)

Draft storefront for **Venom**, the dermatologist-founded men's skincare line by Dr. Amit Om. Single-product launch site for the **Venom Face Mist** (Blue Light Defense Complex).

- Static HTML/CSS + ~40 lines of JS. No build step.
- Built directly to the **Venom Brand Guidelines 2026** — colors, IBM Plex Sans Condensed + Inter type, "venOM" wordmark/emblem (rebuilt as SVG), and voice ("Less noise. More results.").
- Primary CTA is **early-access email capture**; the buy flow swaps to a Stripe Payment Link at launch (see `js/main.js`).

> Draft for review. Photography is from the brand deck (Pinterest-sourced, **not licensed for commercial use**) and must be replaced with real product photography before this goes live to sell.
